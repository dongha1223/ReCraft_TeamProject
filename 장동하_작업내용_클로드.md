- Stage 1: NPC존 (적 없음, 표지판만 존재 — 추후 상점/강화 공간 예정)

- Stage 3: 신규 맵 (Stage 2 지형 복제 기반, 오크 + 마법사 혼합)

**생성한 파일**

| 파일 | 역할 |
|------|------|
| `Assets/Scripts/Core/StageManager.cs` | 스테이지 전환·적 카운트 총괄 싱글턴 |
| `Assets/Scripts/Core/FadeManager.cs` | UIToolkit 기반 FadeIn/FadeOut·게임클리어 UI |
| `Assets/Scripts/Stage/SignpostController.cs` | 표지판 상호작용 (알파·F키·이벤트 구독) |
| `Assets/Scripts/Stage/EnemySpawner.cs` | 스테이지 활성화 시 적 등록 및 재방문 복구 |
| `Assets/UI/GameClearUI.uxml` / `.uss` | GAME CLEAR 오버레이 + 카운트다운 레이블 |

**핵심 로직**
- `EnemySpawner.OnEnable`: 스테이지 루트 활성화 시 자동 호출 → 하위 EnemyStats 전부 찾아 `ResetStats()` + `RegisterEnemy()`
- `EnemyStats.OnDead()`: 사망 시 `StageManager.OnEnemyDied()` 호출 → 카운터 감소
- `StageManager.ActivateStage()`: `_aliveEnemyCount = 0` 초기화를 `SetActive(true)` **전**에 실행 (순서 중요)
- `CheckAllEnemiesDeadNextFrame()`: SetActive 직후 한 프레임 대기 후 AllEnemiesDead 검사 → 이벤트 발행
- `SignpostController`: `OnAllEnemiesDead` 이벤트 구독 → 수신 시 알파 50→255, F키 프롬프트 활성화

**게임 클리어 흐름**
- Stage 3 표지판 `_isLastStage = true` 설정
- F키 → `TriggerGameClear()` → FadeOut → GAME CLEAR 표시 → FadeIn → 10초 카운트다운 → FadeOut → Stage 1 복귀 → FadeIn

**해결한 버그**
1. 게임 시작 시 Stage 3에서 시작하는 버그
   → `GameObject.Find()`는 비활성 오브젝트를 탐색 못 함. Stage2·3를 에디터에서 비활성화하여 해결
2. 표지판이 적 처치 전에 활성화되는 버그
   → `CheckAllEnemiesDeadNextFrame` 코루틴으로 다음 프레임에 검사하도록 수정
3. Stage 1→2 전환 시 오류 (UnassignedReferenceException)
   → Inspector에서 직접 할당 후 해결, 불필요한 자동 탐색 코드 전부 제거

## 2026-04-07

### 표지판 [F] 상호작용 버그 수정

**문제**
- 적이 0마리인 상태에서 표지판�� 접근해도 [F] 프롬프트가 뜨지 않고 다음 스테이지로 이동도 안 됨

**원인 분석 (3중 복합 버그)**
1. **`PlayerInteractor` 컴포넌트가 Player에 없었음** — 핵심 원인. 플레이어가 상호작용 탐지 자체를 전혀 안 하고 있었음
2. **씬의 Signpost 3개 모두 Collider2D 없음** — `Physics2D.OverlapCircle`이 탐지 불가
3. **`OverlapCircleNonAlloc` 사용** — Unity 6에서 deprecated. ContactFilter2D 없이 호출 시 일부 환경에서 트리거 콜라이더 미감지

**수정 내용**
- `Player` GameObject에 `PlayerInteractor` 컴포넌트 추가 (씬 저장)
- `Signpost`, `Signpost_Copy` × 2 에 `BoxCollider2D (isTrigger=true, size 1×1)` 추가 (씬 저장)
- `PlayerInteractor.FindClosest()`: `Physics2D.OverlapCircleNonAlloc` → `Physics2D.OverlapCircle(ContactFilter2D)` 교체
  - `ContactFilter2D.useTriggers = true`, `useLayerMask = false` (모든 레이어 포함)
- 거리 비교: `Vector2.Distance` (sqrt) → `sqrMagnitude` 로 교체 (매 프레임 불필요한 sqrt 제거)
